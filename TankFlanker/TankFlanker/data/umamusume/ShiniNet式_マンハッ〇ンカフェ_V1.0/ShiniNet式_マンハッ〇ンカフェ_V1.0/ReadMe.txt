■モデルについて
　
　・マグカップの使い方
　
　　１．「マグカップ.pmx」をMMDに読み込んで本体モデルの「右手ダミー」ボーンへ外部親登録する。
　　２．モデル本体へ「マグカップを右手で握る.vmd」を読み込むと、腕と指のポーズがイイ感じになります。
　　３．マグカップのモーフで湯気を出したり揺らしたり、中身を無くしたりして遊んでください。
　　
　　以上。
　　
　・シェーダーセットアップ済みPMXの使い方（マンハッ〇ンカフェ_V1.0 withUUS.pmx）
　　手っ取り早く本家のようなトゥーンシェーダーを再現したい人に非常におすすめです。
　　※MME(MikuMikuEffect)が既にMMDへ導入されていること。
　　
　　１．「マンハッ〇ンカフェ_V1.0 withUUS.pmx」をMMDにドラッグ＆ドロップ。
　　２．以上。パッキリとした影が描画されます。
　　　
　　（シェーダーコントローラーの使い方）
　　　カフェちゃん本体に組み込まれたモーフやボーンを操作することで、
　　　シェーダーの描画をコントロールすることができます。
　　　
　　　・セルフ影ON：モーフONで自身や他オブジェクトからの落ち影が描画されるようになります。
　　　　　　　　　　※セルフ影のジャギーが気になる人はExcellentShadow2フォルダ配下の
　　　　　　　　　　「ExcellentShadow.x」をMMDへドラッグ＆ドロップしてください。影がなめらかになります。
　　　・影範囲縮小：モーフによって影の描画範囲を制御します。
　　　・影範囲拡大：〃
　　　・Shader光RGB：ボーンのXYZ座標の移動でモデルを色調することができます。
　　　・Shader影RGB：〃
　　　
　　（使用エフェクト）
　　　・UmaUmaShader：Water-blue(@YATTA____ZE)様
　　　・ExcellentShadow2：そぼろ(sovoro_mmd)様
